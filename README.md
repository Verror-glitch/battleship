# battleship
tugas project coding 2025

cara memainkan/pakai :
1, jalankan program pythonnya terlebih dahulu.
2, ketika programnya sudah dijalankan dan dimulai, munculah simbol "O" di board 5×5.
3, komputer akan menyembunyikan battleship di row dan column secara random.
4, di game tersebut diberikan 4 kesempatan bermain.
5, setiap giliran, ketik:
- Guess Row (0–4)
- Guess Col (0–4)
6, ketika row dan column tersebut cocok dengan ship tersebut berarti menang.
7, jika tidak maka bertanda, simbol "O" akan berubah menjadi "X", dan board akan berubah.
8, jika 4 kali tebakan salah maka kalah atau "game over" dan posisi kapal akan terlihat.
